# 🌤️ Weather App

A Python application that shows real-time weather for any city in the world.

## What it does
- Shows current temperature in Celsius
- Shows feels like temperature
- Shows humidity percentage
- Shows weather condition

## Technologies Used
- Python
- OpenWeatherMap API
- Requests library

## How to run
1. Install requests: pip install requests
2. Add your API key in the code
3. Run: python weather.py
4. Enter any city name
